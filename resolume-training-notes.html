I recently reviewed the Resolume Training pages. These are a few notes on things I found relevant or want to explore further:

Colors

  • Bright.Contrast effect - adjust color with Brightness and Contrast controls
  • Levels effect - adjust color with Base, Midpoint, and Ratio controls
  • Hue Rotate effect - change the color of a clip with Hue and Saturation controls
  • Saturation effect - adjust the saturation of a clip
  • Toggle RGBA channels using buttons on the Clip panel
  • Add Subtract effect - adjust R, G, and B values individually
  • Color Pass effect - allow some colors in the clip to pass while others are desaturated
  • Colorize effect - applies a single color to the whole clip
